주요 업무

전반적인 게임플레이 파트와 적 NPC 로직을 구현했고 추가적으로 최적화, 작은 씬들, 레벨 제작을 위한 툴 등을 담당했습니다.

  • 메인 메뉴와 크레딧 씬 구현

Individual project
Used: C++, OpenGL
Source 1: https://github.com/junsujang-digipen/Graphics-project-BasicGraphics
Source 2: https://github.com/junsujang-digipen/Graphic-project-Reflection-Refraction
Source 3: https://github.com/junsujang-digipen/Graphics-project-BV-SP

Academic Projects that implemented varied graphic techniques.

The first project implemented basic graphics techniques such as Basic meshes, Lighting, Shadow, Fog, Noise, Curves, Tessellation shader, Geometry shader.

The second project implemented texture mapping, sky box, reflection and refraction techniques.

The third project implemented an object loader using the Assimp library, calculated bounding volumes using different methods, and implemented spatial partitioning using Octree and BSPtree methods.
