Spy The Man


프로젝트 소개

게임 이름: Spy The Man 팀원: 장성원, 장준수, 김상범, 최동아 역할: 테스트 매니저, AI 프로그래머 프로젝트 소요 시간: 9달 (2021.09 ~ 2022.06) 사용된 언어: C++ 사용된 툴: Visual Studio 2019 소스 코드: Spy The Man Source Code

대학에서 진행했던 4인 팀 프로젝트입니다. 횡스크롤기반 2D 퍼즐 게임으로 맵에 배치되어 있는 오브젝트와 레벨마다 주어지는 스킬들을 이용해 목표 지점까지 도달하는 게임입니다. C++과 OpenGL을 이용한 자체엔진으로 제작되었고 엔진과 AI 작업에 주로 기여하였습니다.

게임 영상:

주요 업무

인풋 시스템 같은 엔진 기능들의 구현을 담당했고 NPC들의 로직과 NPC들을 관리하는 시스템 담당했습니다.

  • 카메라, 인풋 시스템, 사운드 매니저, 콜리전, 씬 매니저 등 엔진 기능 구현
  • 적대적, 중립 NPC들의 기본 행동 설계 및 구현
  • NPC의 시각, 청각 감지 기능 구현
  • NPC 길 찾기 구현
  • NPC들에게 임의의 작업을 자체적으로 판단하여 지시하는 최상위 관리자 AI 구현
  • 임의의 작업을 받아 실행 시킬 수 있는 커스텀 FSM 구현
  • 플레이어의 스킬과 NPC 상태 제작을 간편화 하기 위한 템플릿 클래스 구현
