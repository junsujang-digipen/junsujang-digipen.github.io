Spy The Man


프로젝트 소개

게임 이름: Spy The Man 팀원: 장성원, 장준수, 김상범, 최동아 역할: 테스트 매니저, AI 프로그래머 프로젝트 소요 시간: 9달 (2021.09 ~ 2022.06) 사용된 언어: C++ 사용된 툴: Visual Studio 2019 소스 코드: Spy The Man Source Code

대학에서 진행한 4인 팀 프로젝트입니다. C++과 OpenGL을 이용한 자체엔진으로 제작되었고

The objective of the game is to reach the goal point while avoiding hostile NPCs by interacting with objects and using the player's skills. There are about 20 stages in total.

We used our own engine, and my main tasks were engine creation and NPC AI implementation.

For the engine, I implemented the basic framework and features of the engine such as cameras, collisions, inputs, windows, and scene managers.

And for the AI, I implemented a head AI that manages all other NPCs to make it look like NPCs are interacting with each other, a task system that allows NPCs to take on unscheduled tasks, and path finding function for NPCs.
