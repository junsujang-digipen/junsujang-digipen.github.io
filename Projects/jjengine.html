프로젝트 소개

엔진 이름: JJEngine 팀원: 서준우, 장준수, 김대현 프로젝트 소요 시간: 3달 (2023.05 ~ 2023.07) 사용된 언어: C++ 사용된 툴: Visual Studio 소스 코드: https://github.com/junsujang-digipen/JJEngine

대학 프로젝트 수업을 위해 3인으로 진행한 게임 엔진 프로젝트입니다. 플래피 버드의 간단한 게임 시스템 구현을 목표로 하여 엔진 기능을 구현하였습니다. C++을 기반으로 하였고 OpenGL, Mono, ENTT, IMGUI 등의 라이브러리가 엔진 기능을 위해 사용되었습니다. Mono를 이용해 C# 기반의 스크립트 기능을 제공하고, IMGUI 기반의 에디터가 제공됩니다. 전체적으로 유니티를 모티브로 구현되었습니다.

엔진 영상:

주요 업무

주로 담당한 작업은 시리얼 라이저, 에디터 기능, 쉐이더 프리프로세서, 에셋 매니저 등 그래픽 엔진과 스크립트 엔진을 제외한 부분에 주로 관여했습니다. 부가적으로 에디터 구조와 편의 기능, 오브젝트 컴포넌트, 엔진 기능들을 에디터에서 사용 할 수 있도록 하는 일을 담당했습니다.

  • YAML 라이브러리를 사용해 씬과 오브젝트 컴포넌트들을 저장, 로드 할 수 있는 시리얼라이저 구현
  • regex 함수를 이용해 하나의 파일에 작성된 쉐이더들을 분할, 재배치 하여 쉐이더 프로그램을 만들어주는 기능 구현
  • 에디터에서 에셋 파일과 폴더 구조를 보고, 상호작용 할 수 있도록 에셋 브라우저 패널 구현
  • 씬 내부에 생성된 오브젝트들의 계층구조를 볼 수 있는 패널 구현
  • 컴포넌트 패널 구현
