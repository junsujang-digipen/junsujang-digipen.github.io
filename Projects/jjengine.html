Project Cube


프로젝트 소개

엔진 이름: JJEngine 팀원: 서준우, 장준수, 김대현 프로젝트 소요 시간: 약 3달 (2023.05 ~ 2023.07) 사용된 기술: C++, ImGui, 구문분석, ENTT 사용된 툴: Visual Studio 소스 코드: https://github.com/junsujang-digipen/JJEngine

상용 엔진을 통해 게임 제작을 진행하면서, 엔진과 에디터의 구현을 좀 더 깊이 이해하고 싶었고, Unity와 같은 게임 에디터, 툴을 어떤 방식으로 구현할 수 있는지 내부적으로 어떤 구조로 구성되는지를 체험적으로 익히고자 했습니다. 전반적으로 유니티 에디터의 레이아웃을 레퍼런스로 삼았고, 사용자가 직접 오브젝트를 배치하고 동작을 제어할 수 있는 “에디터 기반의 엔진”을 구현하여 플래피 버드 같은 간단한 게임 시스템 구현할 수 있는 수준을 목표로 삼았습니다.

엔진 영상:

3개월간 3인 팀으로 협업하며, 시리얼 라이저, 에디터 기능, 쉐이더 프리프로세서, 에셋 매니저 등의 엔진의 에디터 및 리소스 관리 파트를 주도적으로 구현했습니다.

  • YAML을 이용한 씬/오브젝트 직렬화 시스템을 설계하여, 에디터에서 배치한 오브젝트와 컴포넌트 구성을 파일로 저장·로드할 수 있도록 했습니다.
  • 또한, 정규식을 활용해 하나의 파일에 작성된 멀티 쉐이더를 파싱하고, 자동으로 분리·컴파일하는 쉐이더 프리프로세서를 구현했습니다.
  • 에셋 브라우저, 하이어라키 패널, 컴포넌트 패널 등 에디터 인터페이스를 직접 설계하여 사용자가 엔진 기능을 시각적으로 다룰 수 있게 했습니다.
